# Roadmap

## Milestone 1: Playable Local Core

- Extract the cinematic renderer and authored visual assets.
- Preserve the persistent local card loop.
- Keep the local build, contracts, fixtures, and browser smoke proof runnable.
- Make first public contribution flow obvious and proof-backed.

## Milestone 2: Authorable ValenGateway

- Add a public hook runner.
- Add `manage_valen_hooks`.
- Add tool-wrapper registration with `bodyMapping`.
- Add direct-hook and queue-and-poll examples.
- Keep examples local-first and separate from private hosted infrastructure.

## Milestone 3: Optional Adapters

- Add local Codex, Cursor, OpenClaw, and Hermes adapter examples.
- Document a future hosted adapter without embedding private service behavior.
